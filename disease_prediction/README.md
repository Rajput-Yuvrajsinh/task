# Diabetes Prediction System

## Overview
A Machine Learning based system to predict Diabetes (Yes / No)
using the Pima Indians Diabetes Dataset.

## Dataset
Publicly available diabetes dataset.

## ML Models
- Logistic Regression
- Decision Tree
- Random Forest (Final Model)

## Evaluation Metrics
- Precision
- Recall
- F1-score

## Explainability
Feature importance using Random Forest.

## API Endpoint
POST /predict

## Run API
uvicorn api:app --reload
